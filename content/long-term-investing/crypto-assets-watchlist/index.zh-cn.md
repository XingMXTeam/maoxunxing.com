## 我怎么看加密资产

加密资产我会看，但不会放太重。

原因很简单：它确实有新东西，比如稳定币、链上金融、DeFi、AI Agent、全球化交易这些方向。但它也很不稳定，涨跌都很夸张，很多项目今天很热，过一段时间就没人提了。

所以我不会把它当成家庭资产的地基。

对我来说，它更像一个小仓位的观察区。买一点，是为了让自己跟着这个行业看下去；仓位不大，是为了不让它影响家里的现金流和生活节奏。

## 几个标的分别是做什么的

### ETH：链上应用的底层资产

ETH 可以理解成以太坊生态里的基础资产。

很多 DeFi、稳定币、NFT、链上应用都和 Ethereum 有关系。你在链上转账、交易、调用合约，很多时候都要用 ETH 付手续费。ETH 也可以拿去质押，获得一点质押收益。

我看 ETH，主要不是看它短期能涨多少，而是看链上应用以后还会不会继续存在。如果链上金融和稳定币还在发展，ETH 就很难完全被绕开。

它的问题也很明显：手续费、扩容、Layer2 分走价值、其他公链竞争，都会影响 ETH 的表现。

### LINK：给链上应用喂数据

LINK 背后是 Chainlink。

区块链自己不知道现实世界发生了什么。比如 ETH 多少钱、黄金多少钱、某个抵押品价格有没有跌破线，这些都需要外部数据。

Chainlink 做的就是把这些数据送到链上。DeFi 借贷、清算、衍生品，很多地方都会用到这种数据服务。

我关注 LINK，是因为这个需求比较真实。但代币本身能不能把这个需求转成长期价值，还要继续看。

### AAVE：链上的借贷市场

AAVE 做的是 DeFi 借贷。

有人把资产存进去赚利息，有人抵押资产借钱，中间靠智能合约自动执行规则。你可以把它理解成链上的资金市场。

它比很多纯故事项目好理解，因为确实有人用，也确实有收入。

但风险也不少：合约漏洞、抵押品暴跌、清算踩踏、监管变化，都会影响它。

### stETH：带质押收益的 ETH

stETH 是 Lido 的 ETH 质押凭证。

你把 ETH 拿去质押，Lido 给你 stETH。它大体上代表你质押进去的 ETH，同时会反映质押收益。

如果本来就打算长期拿 ETH，stETH 可以理解成“多拿一点质押收益”的方式。

但它比直接拿 ETH 多了一层风险：Lido 协议风险、流动性折价、DeFi 组合风险，都要算进去。

### wBETH：币安里的 ETH 质押资产

wBETH 和 stETH 类似，也是 ETH 质押相关资产。

区别是它更依赖币安生态。如果你主要在 Binance 里操作，它会比较方便。缺点也很直接：平台依赖更强，去中心化程度更低。

所以它不是比 stETH 更好或更差，而是适用场景不一样。

### BNB：币安生态的代币

BNB 和 ETH 不太一样。

ETH 更像开放公链的基础资产，BNB 更像 Binance 这个大生态里的代币。它和交易平台、BNB Chain、手续费、Launchpad、生态应用都有关系。

BNB 的优点是背后有真实用户和商业生态。缺点也在这里：它对 Binance 的依赖很高，平台风险和监管风险都比较集中。

所以我不会把 BNB 当成 ETH 的替代品。它更像交易平台生态的一部分。

### VIRTUAL：AI Agent 方向的早期项目

VIRTUAL 背后是 Virtuals Protocol。

我关注它，是因为它把 AI Agent 和 Crypto 放到了一起。

如果以后 AI Agent 不只是聊天工具，而是能在网上提供服务、收钱、分账、拥有自己的身份和钱包，那区块链可能会有用武之地。

但现在这个方向还很早。真实用户、真实收入、代币价值怎么捕获，都还没完全证明。

所以 VIRTUAL 适合观察，不适合上头。

### PAXG：链上的黄金

PAXG 比较简单，就是黄金锚定资产。

它不是成长资产，不应该拿来和 ETH、BNB、VIRTUAL 比涨幅。它更像把黄金搬到链上，方便在加密账户里持有、转账、交易。

它的角色偏防守，不是进攻。

### MANA：元宇宙主题资产

MANA 是 Decentraland 的代币，代表虚拟土地、虚拟世界、元宇宙经济这一类方向。

这个故事以前很热，但现实进展没有想象中快。用户、使用频率、商业化，都还需要重新证明。

所以我会把 MANA 放在主题观察里，不会把它当核心资产。

## 我会怎么放

| 标的 | 我怎么理解 | 位置 |
| --- | --- | --- |
| ETH | 链上应用的底层资产 | 加密资产里的核心观察 |
| LINK | 链上数据服务 | 基础设施观察 |
| AAVE | 链上借贷市场 | DeFi 观察 |
| stETH / wBETH | 带质押属性的 ETH | ETH 增强工具 |
| BNB | Binance 生态代币 | 平台生态观察 |
| VIRTUAL | AI Agent + Crypto | 高风险小仓位观察 |
| PAXG | 链上黄金 | 防守资产 |
| MANA | 元宇宙主题 | 主题观察 |

我的原则很简单：

BTC、ETH 这种更基础的，可以小比例长期看。

LINK、AAVE、BNB 这种有实际用途的，可以看它们是不是真的有长期需求。

VIRTUAL、MANA 这种主题性更强的，要更谨慎。故事可以听，但仓位不能跟着故事走。

PAXG 就别幻想暴涨了，它更像黄金。

加密资产最容易让人情绪化，因为涨跌太快。对普通家庭来说，最重要的不是押中哪个币，而是别让一次判断错误影响生活。

真正的地基，还是现金流、现金池、指数基金、房贷管理和家庭支出控制。加密资产可以有，但只能是很小的一层。
