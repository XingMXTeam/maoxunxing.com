## 一、作家人设模板

来源：基于 1688 社媒人设模板仿写

### 核心画像

**身份**：穿梭于文字宇宙的故事建筑师
**账号昵称**：笔锋先生/笔锋小姐

**性格特征**：
- 敏锐洞察：捕捉生活褶皱里的故事种子
- 诗意通透：用比喻编织语言，让平凡场景绽放哲思
- 温暖陪伴：文字世界的领航员，也是读者情绪的树洞
- 持续进化：每本书都是新自我

### 视觉与语言调性

**视觉**：墨蓝（深邃感）+ 暖黄（人文温度），留白 + 手写批注风格

**语言**：
- 文学化口语：将"写作技巧"转化为"故事炼金术"
- 金句驱动：每篇结尾抛出可截图传播的短句
- 场景化叙事：描述创作过程时构建画面感

### 内容栏目规划

| 栏目 | 内容示例 | 目的 |
|------|----------|------|
| 灵感速递 | 暴雨天的便利店监控录像故事 | 激发观察力 |
| 创作解密 | 如何让反派角色不沦为工具人 | 专业技巧 |
| 笔锋夜话 | 当 AI 开始写诗：人类最后的浪漫 | 思想引领 |
| 读者共笔 | 为分手恋人续写结局 | 互动共创 |

### 金句库

- "故事是人类对抗虚无的锚点，而我愿做那个抛出锚绳的人。"
- "每个标点都是呼吸的间隙，我在句与句的留白处藏了整个宇宙。"
- "别怕故事太小，最微弱的火柴也能点燃整片草原的想象。"

### 执行要点

1. 通过连载进度条、创作手记展现创作过程
2. 将书籍主题与热点结合
3. 每月固定日期开设"故事诊疗所"

## 二、PixPin - 截图贴图效率工具

来源：https://pixpin.cn/

截图 + 贴图的组合工具，支持 Windows 和 macOS。

### 核心功能

| 功能 | 说明 |
|------|------|
| 区域截图 | 灵活的屏幕区域截取 |
| 长截图 | 长页面自动拼接 |
| GIF 录制 | 动态影像录制 |
| 贴图 | 截图转为屏幕悬浮参考物，支持全局快捷键 |
| OCR | 图文转换 |
| 标注 | 图形标记与注释 |
| UI 元素识别 | 自动识别界面元素 |

**定价**：免费基础版 + 付费订阅制

**亮点**：贴图悬浮功能，比传统截图工具多了一层"参考物"的使用方式，特别适合文档编写和演示制作。

## 三、PixelLab - AI 像素艺术与游戏素材工具

来源：https://www.pixellab.ai/

PixelLab 是一个面向像素艺术、游戏素材和轻量视觉创作的 AI 工具。它适合用来快速生成角色、物体、环境、UI 元素和动画帧，也可以作为小游戏、短视频视觉实验、独立开发原型的素材生产工具。

### 核心功能

| 功能 | 说明 |
|------|------|
| 文生像素图 | 根据文字描述生成像素风角色、物体和环境 |
| 图像编辑 | 通过指令修改图片，例如添加、替换或移除元素 |
| 角色方向图 | 生成 8 方向角色精灵，适合游戏角色移动素材 |
| 动画生成 | 根据参考图和动作描述生成动画帧 |
| UI 素材 | 生成血条、按钮、背包格、菜单等游戏 UI 元素 |
| 浏览器编辑 | 集成 Pixelorama，可以在浏览器里完成编辑和 AI 生成 |
| API 接入 | 可通过 API 接入素材生成流程，适合自动化工作流 |

### 使用场景

- 给小游戏、H5、Phaser 项目快速做像素风资产
- 为 AI 互动故事、短视频、图文内容生成统一风格素材
- 做产品原型时快速补齐角色、地图、图标和 UI 占位图
- 作为 Aseprite / Pixelorama 工作流里的 AI 辅助工具

**亮点**：它不是泛用型图片生成工具，而是更聚焦像素艺术和游戏资产。对个人开发者来说，价值在于把"想做一个游戏/互动内容，但缺美术素材"这个卡点往前推了一步。
