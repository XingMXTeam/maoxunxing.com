## 一、内容人设模板

来源：基于 1688 社媒人设模板仿写。

这部分更像账号定位和内容调性的参考，不是固定要照抄的人设。适合在做公众号、小红书、视频号或个人 IP 页面时，用来快速找到一种稳定表达方式。

### 作家人设示例

**身份**：穿梭于文字宇宙的故事建筑师  
**账号昵称**：笔锋先生 / 笔锋小姐

**性格特征**：
- 敏锐洞察：捕捉生活褶皱里的故事种子
- 诗意通透：用比喻编织语言，让平凡场景绽放哲思
- 温暖陪伴：文字世界的领航员，也是读者情绪的树洞
- 持续进化：每本书都是新自我

### 视觉与语言调性

**视觉**：墨蓝（深邃感）+ 暖黄（人文温度），留白 + 手写批注风格

**语言**：
- 文学化口语：将“写作技巧”转化为“故事炼金术”
- 金句驱动：每篇结尾抛出可截图传播的短句
- 场景化叙事：描述创作过程时构建画面感

### 内容栏目规划

| 栏目 | 内容示例 | 作用 |
|------|----------|------|
| 灵感速递 | 暴雨天的便利店监控录像故事 | 激发观察力 |
| 创作解密 | 如何让反派角色不沦为工具人 | 展示专业技巧 |
| 笔锋夜话 | 当 AI 开始写诗：人类最后的浪漫 | 输出观点 |
| 读者共笔 | 为分手恋人续写结局 | 互动共创 |

### 金句库

- “故事是人类对抗虚无的锚点，而我愿做那个抛出锚绳的人。”
- “每个标点都是呼吸的间隙，我在句与句的留白处藏了整个宇宙。”
- “别怕故事太小，最微弱的火柴也能点燃整片草原的想象。”

### 执行要点

1. 通过连载进度条、创作手记展现创作过程
2. 将书籍主题与热点结合
3. 每月固定日期开设“故事诊疗所”

## 二、截图与标注：PixPin

链接：[PixPin](https://pixpin.cn/)

PixPin 是一个截图 + 贴图工具，支持 Windows 和 macOS。它适合做教程、产品分析、文章配图、演示稿和开发沟通。

### 核心功能

| 功能 | 说明 |
|------|------|
| 区域截图 | 灵活截取屏幕区域 |
| 长截图 | 长页面自动拼接 |
| GIF 录制 | 录制动态操作过程 |
| 贴图 | 把截图悬浮在屏幕上，作为参考物 |
| OCR | 图文转换 |
| 标注 | 加箭头、框线、说明文字 |
| UI 元素识别 | 自动识别界面元素 |

**我会怎么用**：写教程、拆产品页面、做小红书图文、整理网页参考时，PixPin 的贴图能力比普通截图工具更顺手。

## 三、AI 视觉与像素素材：PixelLab

链接：[PixelLab](https://www.pixellab.ai/)

PixelLab 是一个面向像素艺术、游戏素材和轻量视觉创作的 AI 工具。它适合快速生成角色、物体、环境、UI 元素和动画帧，也适合小游戏、短视频视觉实验、独立开发原型。

### 核心功能

| 功能 | 说明 |
|------|------|
| 文生像素图 | 根据文字描述生成像素风角色、物体和环境 |
| 图像编辑 | 通过指令添加、替换或移除元素 |
| 角色方向图 | 生成 8 方向角色精灵，适合游戏角色移动素材 |
| 动画生成 | 根据参考图和动作描述生成动画帧 |
| UI 素材 | 生成血条、按钮、背包格、菜单等游戏 UI 元素 |
| 浏览器编辑 | 集成 Pixelorama，可以在浏览器里编辑和生成 |
| API 接入 | 可接入素材生成流程，适合自动化工作流 |

### 使用场景

- 给小游戏、H5、Phaser 项目快速做像素风资产
- 为 AI 互动故事、短视频、图文内容生成统一风格素材
- 做产品原型时快速补齐角色、地图、图标和 UI 占位图
- 作为 Aseprite / Pixelorama 工作流里的 AI 辅助工具

**我会怎么用**：它不是泛用型图片生成工具，而是更聚焦像素艺术和游戏资产。对个人开发者来说，它能把“想做互动内容，但缺美术素材”的卡点往前推一步。

## 四、图片素材：Unsplash

链接：[Unsplash](https://unsplash.com/)

Unsplash 更适合做文章封面、生活方式配图、氛围图和视觉参考。它不是我每次都会用的工具，但适合在需要“真实照片质感”时找素材。

### 精选照片

**La Jolla 海滩奔跑**
- 摄影师：Frank McKenna (@frankiefoto)
- 地点：La Jolla, San Diego, United States
- 相机：Nikon D4
- 适用场景：封面图、海滩主题背景、生活方式类配图

### 推荐摄影师

| 摄影师 | 风格 | 照片数 |
|--------|------|--------|
| Damian Denis (@dwade) | 多学科设计与摄影，华沙 | 5 |
| Aleks Dahlberg (@aleksdahlberg) | 风景与自然、高山、探险、黑白构图，新西兰 | 131 |

### 莫奈风格素材

搜索关键词 `monet` 可获得与 Claude Monet、印象派、油画、花园相关的照片和插画素材。

### 使用须知

- Unsplash 照片通常可免费用于商业和非商业用途
- 不强制注明出处，但我更倾向于保留摄影师信息
- 不要把照片原样打包售卖，也不要上传到竞品图库平台
