## 2. Screenshots and annotations: PixPin

Link: [PixPin](https://pixpin.cn/)

PixPin is a screenshot and pin-to-screen tool for Windows and macOS. I use it for tutorials, product teardown notes, article images, slide references, and development communication.

Useful features:

- Area screenshots
- Long screenshots
- GIF recording
- Pin screenshots to the screen as floating references
- OCR
- Annotations
- UI element recognition

Why it matters: the pinned screenshot feature makes it more useful than a normal screenshot tool when writing docs or comparing interfaces.

## 3. AI pixel art and game assets: PixelLab

Link: [PixelLab](https://www.pixellab.ai/)

PixelLab is an AI tool for pixel art, game assets, and lightweight visual creation. It can generate characters, objects, environments, UI elements, and animation frames.

Useful features:

- Text-to-pixel-art generation
- Image editing through instructions
- 8-direction character sprites
- Animation generation
- Game UI assets such as health bars, buttons, inventory slots, and menus
- Browser-based editing with Pixelorama
- API access for asset workflows

Where I would use it:

- Small games, H5 projects, or Phaser prototypes
- AI interactive stories and visual experiments
- Placeholder assets for product prototypes
- Aseprite / Pixelorama-assisted workflows

Why it matters: it is not just a general image generator. It is more focused on pixel art and game assets, which helps solo builders move past the “I want to build this, but I have no art assets” bottleneck.

## 4. Photo assets: Unsplash

Link: [Unsplash](https://unsplash.com/)

Unsplash is useful when I need real-photo texture for article covers, lifestyle visuals, atmosphere images, or visual references.

Example use cases:

- Article covers
- Background images
- Visual references for mood and composition
- Lifestyle or travel-related visuals

Usage notes:

- Unsplash photos are generally free for commercial and non-commercial use
- Attribution is not required, but I prefer to keep photographer information when possible
- Do not resell original photos or redistribute them as a competing image library
