## VS Code Forward/Backward

Ctrl + -: Forward (equivalent to Ctrl + [ in WebStorm)
Ctrl + - + Shift: Backward (equivalent to Ctrl + ] in WebStorm)

## Sublime Multi-line Editing

Hold down the Option key and drag with the mouse
