## GitHunt: GitHub 热门项目发现

来源：https://kamranahmed.info/githunt

| 项目 | 详情 |
|------|------|
| 作者 | Kamran Ahmed（roadmap.sh 创建者） |
| 类型 | React 应用 + Chrome 扩展 |
| 许可 | MIT |
| 贡献者 | 21 人，197 次提交 |
| 项目年龄 | 8 年以上 |

### 功能

- 按时间段浏览每周热门项目，支持历史翻页
- 按编程语言筛选，记忆语言偏好
- 列表/网格双视图
- 可添加 GitHub Token 避免 API 限流
- Chrome 扩展替换新标签页为热门项目展示

### 使用场景

- 发现新的热门开源项目
- 追踪特定语言的趋势仓库
- 替代 GitHub Trending（更好的筛选和历史回溯）

## ZenMux: 企业级大模型聚合网关

来源：https://zenmux.ai/

定位为全球首个支持保险赔付机制的企业级大模型聚合平台。

### 核心特色

| 特性 | 说明 |
|------|------|
| 统一 API | 一套接口调用 87+ 模型（OpenAI、Anthropic、Google、DeepSeek 等） |
| 双协议 | 兼容 OpenAI 与 Anthropic 协议标准 |
| 智能路由 | 根据任务特征自动权衡效果与成本 |
| AI 保险（首创） | 幻觉、效果波动、高延迟自动检测，隔日理赔 |
| 全链路可观测 | 调用日志、成本核算、用量统计、性能追踪 |
| 多供应商容灾 | 自动故障切换，全球节点低延迟 |

**定价**：订阅制 + 按量计费，支持信用卡和支付宝。

## OpenClaw: AI 网关命令行工具

来源：https://docs.openclaw.ai/web

```bash
openclaw dashboard    # 通过界面聊天
openclaw gateway restart  # 重启 gateway
openclaw tui          # 通过命令行开始对话
```

## ChatGPT 应用内购买被风控解决方法

**问题**：ChatGPT 应用内无法购买（订阅/升级），账号被风控。

**解决步骤**：
1. 国内客服无法处理此问题
2. 联系**美区客服**
3. 提交申请后约 **48 小时**解除风控
