这周 AI 圈最热闹的，还是 OpenAI 和英伟达。

OpenAI 这边，GPT-5.5 Pro 不只是能力指数刷了新高，还参与解决了一个困扰数学界 60 年的难题。另一边，英伟达也没闲着，直接拿出 Nemotron-3 Nano Omni，主打端侧、多模态和超高推理效率。

更现实的一幕是：数千台通用人型机器人，已经准备正式进入物流仓库干活了。AI 不只是写代码、生成图片，它开始真正走进物理世界。

## 1. 核心大模型发布与技术迭代

### OpenAI GPT-5.5 Pro 来了，而且这次很猛

GPT-5.5 Pro 在 Epoch 能力指数里拿到了 **159 分**，创下历史新高。

更值得注意的是，它已经协助研究人员解决了一个由数学家 Erdős 提出、悬而未决 60 年的数学难题。

这件事的意义不只是“模型又变强了”，而是 AI 开始真正进入顶尖学术研究现场，不再只是辅助写写代码、整理资料。

（来源: <https://x.com/OpenAI/status/2049182118069358967>）
（来源: <https://x.com/EpochAIResearch/status/2049186851844771888>）

### 英伟达发布 Nemotron-3 Nano Omni

英伟达这次发布的是一款 30B 参数的开源模型，采用 Mamba-Transformer 混合架构，上下文长度达到 **256K**。

它的重点不是单纯堆参数，而是把原生音频、长视频、百页文档这些复杂输入，都放进一次处理流程里。

更夸张的是，推理吞吐量相比同类模型提升了 **9 倍**。

说白了，英伟达这次是在告诉大家：端侧全模态模型，也可以又快又能打。

（来源: <https://x.com/NVIDIAAI/status/2049159441870717428>）
（来源: <https://x.com/mervenoyann/status/2049216818150015255>）

### Google Translate 20 周年，顺手加了 AI 口语练习

Google Translate 已经 20 岁了，用户规模超过 **10 亿**。

这次 Google 还上线了新的 AI 发音练习功能，可以实时分析你的口语表现，并给出反馈。

这其实挺有代表性：AI 学习工具正在从“帮你翻译”，变成“陪你练习”。

（来源: <https://x.com/Google/status/2049194510752559426>）
（来源: <https://x.com/NewsFromGoogle/status/2049194438505767387>）

## 2. AI 编程与开发者工具

### Agent 模式开始全面进入办公和开发工具

这周有一个明显趋势：Agent 不再只是单独的聊天机器人，而是在往各种工具里钻。

Microsoft Outlook 现在已经支持 Agent 模式，可以帮你自动分拣邮件、重新安排日程。

OpenAI Codex 也支持把现有代码库一键升级到 GPT-5.5 架构。

Google 则在 GitHub 上线了官方 Agent 技能库。

这些变化放在一起看，其实说明一件事：以后很多工具都不会只是“你点一下它动一下”，而是你给个目标，它自己往前跑。

## 4. 机器人与物理 AI

### 人型机器人真的要去仓库上班了

数千台 RobotEra L7 人型机器人将进入 10 多个物流中心，执行分拣任务。

另外，集成 Agent 软件的 Reach Mini 机器人，也已经开始向开发者交付。

这件事很现实：机器人不再只是展会上跳舞、做演示，而是开始进入仓库、物流这些高重复、高体力消耗的场景。

对企业来说，这是效率提升。
但对一部分体力劳动者来说，这也确实是职业安全警报。

## 5. 行业观点、社会影响与财务状况

### 马斯克和 OpenAI 的矛盾还在升级

马斯克继续炮轰 Sam Altman 和 Greg Brockman，指责他们“窃取”非营利组织资产，并在内部交易中谋取巨额股份利益。

这场拉锯已经不只是商业竞争，更像是围绕 OpenAI 身份、治理和利益分配的一场长期战争。

（来源: <https://x.com/elonmusk/status/2048801964457140540>）

### 互联网内容正在被 AI 改写

最新数据显示，目前大约有 **1/3** 的网站内容由 AI 生成。

还有预测认为，到 2026 年底，这个比例可能飙升到 **99%**。

不管这个预测最终准不准，有一点已经很明显：以后互联网上“像人写的内容”，不一定真的是人写的。

内容生态正在变得越来越复杂。

（来源: <https://x.com/AISafetyMemes/status/2049195355305369761>）

### OpenAI、Amazon、微软之间的关系变得更微妙了

OpenAI 宣布与 Amazon 达成深度合作，把模型和 Bedrock 托管 Agent 引入 AWS。

同时，OpenAI 也更新了与微软的协议。微软仍然是首选伙伴，但对 OpenAI IP 的许可变成了非独家。

这意味着什么？

OpenAI 还需要微软，但它也在给自己留更多空间。
