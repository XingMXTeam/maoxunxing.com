# 快速查看 README 和安装 npm 包

---

## **1. 快速查看 README**

通过命令行快速查看 `README` 文件内容：

```shell
readme net
```

---

## **2. 快速安装 npm 包**

使用 `npmi` 快速安装 npm 包：

```shell
npm install -g npmi
```

---

## **3. 说明**

- **`readme net`**：
  - 用于快速查看项目的 `README` 文件内容。
  - 确保相关工具已正确安装并配置。

- **`npmi`**：
  - 是一个简化的 npm 安装工具，提升安装效率。
  - 安装后可以直接使用 `npmi <package-name>` 来安装依赖包。
