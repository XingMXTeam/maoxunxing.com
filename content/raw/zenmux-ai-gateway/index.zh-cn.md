## 简介

ZenMux 定位为**全球首个支持保险赔付机制的企业级大模型聚合平台**，一个账号、一个 API 即可接入多家模型供应商。

## 核心特色

- **统一 API**: 一套接口调用 OpenAI、Anthropic、Google、DeepSeek 等 87+ 模型
- **双协议兼容**: 原生兼容 OpenAI 与 Anthropic 协议标准，无缝对接主流开发工具
- **智能路由**: 根据任务特征自动权衡效果与成本，最优模型分发
- **AI 模型保险服务（首创）**: 针对幻觉、效果波动、高延迟等场景自动检测，隔日理赔
- **全链路可观测**: 调用日志、成本核算、用量统计、性能追踪
- **多供应商容灾**: 自动故障切换，分布式全球节点低延迟加速
- **透明质量保障**: 定期 HLE 降智检测，数据开源公开

## 定价

订阅制 + 按量计费双模式，支持信用卡和支付宝。

## 个人评价

作为模型网关还是不错的，统一 API + 智能路由 + 保险赔付的组合比较有特色。
