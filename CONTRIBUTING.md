# Contributing

See [README.md](README.md) for:

- Local setup (`git submodule update --init`, `npm install`, `hugo server`)
- Architecture (Hugo + `hugo-coder` submodule, layout overrides, `assets/scss` + `static/css/main.css`)
- Algolia: set `ALGOLIA_ADMIN_KEY` for `npm run index-and-send`; CI uses the `ALGOLIA_ADMIN_KEY` repository secret

Code style: Google guides for [HTML/CSS](https://google.github.io/styleguide/htmlcssguide.html) and [JavaScript](https://google.github.io/styleguide/jsguide.html).

Pull requests: rebase onto `master`, one-line summary, optional `Fixes #NN` for [issues](https://github.com/XingMXTeam/maoxunxing.com/issues).
